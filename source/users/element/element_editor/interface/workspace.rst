============
Drawing area
============

The drawing area or workspace, is the area where the graphical representation of the elements is created. 

The drawing area or workspace looks as follow:

.. figure:: /_external/_images/en/qet_element/qet_element_editor_workspace.png
   :align: center

   Figure: QElectroTech element editor workspace

The scrollbar are not appearing at first when the element editor is displayed. Rolling the wheel from 
the mouse to zoom will make the scrollbar appear.
